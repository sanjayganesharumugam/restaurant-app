# Dish Image Upload Guide

This file lists the restaurant dishes and suggested image file names for each dish.

## How to use

1. Add the image files to the `images/` folder.
2. Use the file names shown below, or keep your own names and update `menu.html` accordingly.
3. Prefer high-quality, open-source food photos.
4. Use `.jpg` or `.png` depending on the image format.
